CHANGELOG for 0.3.0 Release
====================================

New Features
------------

### JSON Config System Integration
- **Feature**: Added a lightweight, dependency-free configuration system utilizing JSON format. Supports reading settings from local `./noxe.json` or user home `~/.noxerc` directory.
- **Config Key "tabsize"**: Integrates dynamically with the render engine inside `src/draw.c`. Tab characters (`\t`) are expanded visually to align with tabsize boundaries, and the cursor positioning handles the variable tab size seamlessly.
- **Config Key "mouse"**: Validates boolean config key `mouse` for future integration or scripting capabilities.
- **Config Key "show_line_numbers"**: Allows users to conditionally show/hide the line number gutter column on the left. The cursor rendering automatically adjusts to alignment offsets.
- **Config Key "auto_indent"**: Toggles automatic indentation of whitespace from the previous line when pressing Enter.
- **Command Line Overrides**: Config options can be overridden dynamically at startup using flags like `--tabsize 8` or `--mouse false` or `--show_line_numbers false`.
- **Live Watch & Auto-Reload**: A filesystem checker is built-in. If the configuration file is updated externally while the editor is running, it automatically detects changes, re-parses settings, and flashes a notification on the status bar.
- **Data Validation**: Schema-checking enforces positive number limits on `tabsize` and strict boolean formats on `mouse`, `show_line_numbers`, and `auto_indent` to prevent bad config files from breaking the UI.
- **Syntax Highlight**: Add syntax highlighting for multiple languages like C, C++, Python using regex and nice classic colors

Bug Fixes
---------
- **Fixed Critical 4KB File Insertion Limit**: Corrected the page-splitting logic in `nodes.c` so the editor does not drop characters after reaching the 4,096 character threshold.
- **Fixed Search Memory Leaks**: Replaced dynamic `arena_alloc` calls in search functions with stack buffers, preventing memory exhaustion crashes.
- **Fixed Search-Previous Navigation**: Corrected wrapping behaviour, resolved the index-0 match detection bug, and stopped the cursor from pinning to the first match.
- **Fixed Viewport Scrolling Lock**: Changed the scroll calculations in `draw.c` so the viewport scrolls naturally when the cursor reaches viewport boundaries, rather than locking the cursor to the bottom line of the screen.
- **Fixed Cross-Page Deletion**: Aligned the gap buffer boundary correctly when backspacing across page boundaries so the last character of the previous page is deleted instead of random characters in the middle.
- **Fixed New File Handling**: Corrected startup checks so opening a non-existent file path prints a clean "New file." status bar message rather than a path canonicalization warning.
- **Fixed Auto-Indentation Formatting**: Replaced the cursor-based whitespace counting with a clean buffer scan that correctly preserves tabs and spaces from the previous line and pushes those insertions to the undo stack.
- **Resolved Stack Buffer Overflow Risk**: Eliminated the temporary stack buffer copy in tab-expansion code in `draw.c` to prevent overflows when rendering long lines.

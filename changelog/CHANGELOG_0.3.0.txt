CHANGELOG for 0.3.0 Release
====================================

New Features
------------

### JSON Config System Integration
- **Feature**: Added a lightweight, dependency-free configuration system utilizing JSON format. Supports reading settings from local `./noxe.json` or user home `~/.noxerc` directory.
- **Config Key "tabsize"**: Integrates dynamically with the render engine inside `src/draw.c`. Tab characters (`\t`) are expanded visually to align with tabsize boundaries, and the cursor positioning handles the variable tab size seamlessly.
- **Config Key "mouse"**: Validates boolean config key `mouse` for future integration or scripting capabilities.
- **Command Line Overrides**: Config options can be overridden dynamically at startup using flags like `--tabsize 8` or `--mouse false`.
- **Live Watch & Auto-Reload**: A filesystem checker is built-in. If the configuration file is updated externally while the editor is running, it automatically detects changes, re-parses settings, and flashes a notification on the status bar.
- **Data Validation**: Schema-checking enforces positive number limits on `tabsize` and strict boolean formats on `mouse` to prevent bad config files from breaking the UI.
